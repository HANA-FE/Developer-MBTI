# 개발자 성향 테스트

MBTI 처럼 개발자의 성향을 알아볼 수 있는 테스트 입니다 :)

## Getting Started

1. 아래의 명령어를 수행합니다.

```bash
$ nvm install
$ nvm use
```

2. 아래의 명령어를 수행합니다.

```bash
$ yarn install
```

```bash
$ yarn dev
```
