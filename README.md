# 개발놈 TEST
[개발놈 TEST 바로가기](https://develop.d3vy5nrbquxsy6.amplifyapp.com)

## 🌟 주요 기능

- 사용자 이름 입력
- 12개의 질문을 통한 개발자 성향 테스트
- 실시간 진행률 표시
- 애니메이션 효과가 있는 로딩 화면
- MBTI 기반 결과 분석
- 결과 공유 및 이미지 다운로드 기능
- 반응형 디자인
